# Milestone Plan (Prompts 02–12)

- **02 App scaffold & deployment**: Vite React TS scaffold, CI basics, preview deployment.
- **03 Data ingestion**: robust CSV loaders + Zod schemas + fixtures.
- **04 Pitch/chord core**: 31-EDO model, chord/inversion domain utilities.
- **05 Analyzer MVP**: structural interval extraction + rule checking outputs.
- **06 Generator MVP**: constrained SATB harmonization baseline.
- **07 UI integration**: one-screen editor + analysis panels + controls.
- **08 Keyboard + microtonal UX**: full 31-EDO keyboard interaction and highlighting.
- **09 Audio playback**: per-voice scheduling and audition controls.
- **10 Explainability polish**: pedagogical messaging, evidence linking, severity taxonomy.
- **11 Hardening**: accessibility, performance, edge cases, regression suite.
- **12 Release prep**: docs, examples, QA checklist, release notes.
